# Practica-Github
## Próposito
Este repositorio fue creado para una prueba personal para aprender y entender con Github.
## Descripción
El objetivo principal es explorar las funciones basicas de Github asi como:
- Creación de repositorio
- Editar y subir archivos
- Escribir un README
- Hacer commits
- Usar el historial de versiones
